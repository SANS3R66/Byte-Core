# Byte Core
 simple core server for Supercell games (maybe) on TypeScript

# Install
## dependencies
 node.js

 open terminal in /src/ and
 ```bash
 npm i
 ```

# starting
 just input in the terminal in /src/
 ```bash
 npm start
 ```
